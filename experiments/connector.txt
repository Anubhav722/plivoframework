===================
Call flow examples
===================
1) Receive Incoming call and bind to Application

phone 
	-> freeswitch 
		-> mod_event_socket outbound
				-> telephonie server 
						-> retrieve GET/POST xml service 
                           			(telephonie.freeswitch.handlers.XMLHandler)

2) Make Outgoing call and bind to Application

send GET/POST xml service 
	-> telephonie server
	   (telephonie.freeswitch.handlers.XMLHandler)
					-> mod_event_socket inbound
							-> freeswitch 
									-> phone




===================
Classes hierarchy
===================

telephonie.freeswitch.handlers.EventsocketHandler
	|
	--- telephonie.freeswitch.handlers.XMLHandler
	|
	--- telephonie.freeswitch.handlers.RESTHandler


===================
Description
===================
telephonie.freeswitch.handlers.EventSocketHandler : 
  base class handlers to run commands with freeswitch eventsocket api

telephonie.freeswitch.handlers.XMLHandler : 
  - This class inherits from telephonie.freeswitch.handlers.EventSocketHandler
  - Parse XML data, convert to freeswitch commands and run them
   

telephonie.freeswitch.handlers.RESTHandler : 
  - This class inherits from telephonie.freeswitch.handlers.EventSocketHandler
  - Parse REST request, convert to freeswitch commands and run them
